ARA DATA MANAGER

¿Qué es el modulo Data Manager?

Este módulo del sistema permite el levantamiento de encuesta, haciendo uso de dispositivos móviles android. Para la correcta utilización de este sistema es necesario proveer una encuesta en formato XML, además se deben sincronizar los dispositivos para la recolección de la información; las instrucciones para dicho proceso aparecen a continuación.

¿Cómo funciona?

Los pasos para subir una encuesta en el sistema son:

  • 1. Crear la encuesta.
  • 2. Exportar la encuesta en formato XML.
  • 3. Subir la Encuesta en formato XML.
  • 4. Sincronizar los dispositivos de recolección de datos.
  • 5. Bajar la encuesta en los dispositivos móviles.
  • 6. Levantar las encuestas.
  • 7. Subir las respuestas al servidor.

1. Crear encuesta

Para la creación de la encuesta se puede utilizar algún sistema de creación de encuestas, a continuación utilizaremos Koobotool, sin embargo, es pertinente aclarar que no es el único servicio.
